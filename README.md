# Sentiment Analyzer

## About this project
This is a simple AI-based sentiment analyzer that I built using Python and Flask.  
It takes user input text and tells whether the sentiment is **Positive, Negative, or Neutral**.

I created this project to understand how machine learning models can be used in real applications.

---

## Features
- Analyze text sentiment (positive / negative / neutral)
- Shows confidence score
- Simple and clean user interface
- Background image and styled UI

---

## Technologies Used
- Python
- Flask
- Hugging Face Transformers
- HTML & CSS

---

## How to run this project

1. Install required libraries:
   pip install flask transformers

2. Run the project:
   python app.py

3. Open browser and go to:
   http://127.0.0.1:5000
